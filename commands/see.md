# /see - 查看屏幕

查看当前屏幕，获取屏幕尺寸、识别到的文字数量和简要描述。

## 使用方法

```
/see
```

## 返回信息

- **屏幕尺寸**: width x height
- **文字数量**: 识别到的文字区域数量
- **简要描述**: 屏幕内容的摘要
- **元素列表**: 识别到的 UI 元素（如果有）

## 示例

```
用户: /see

AI: 屏幕 1920x1080
识别到 23 个文字区域

主要元素:
- [窗口] 终端
- [按钮] 确定
- [输入框] 搜索
```

## 注意事项

- 首次使用会自动启动后台服务
- 需要屏幕录制/辅助功能权限（macOS）
- 确保屏幕内容清晰以便 OCR 识别
