# /click - 点击操作

查找并点击屏幕上指定的文字，或点击指定坐标。

## 使用方法

```
/click <text|x> [y]
```

### 参数

**方式 1: 点击文字**
- **text**: 要查找并点击的文字内容

**方式 2: 点击坐标**
- **x**: X 坐标
- **y**: Y 坐标

## 示例

```
# 点击文字
/click 确定
/click 提交
/click 关闭

# 点击坐标
/click 100 200
```

## 返回信息

- 是否成功找到并点击
- 点击位置（坐标）
- 找到的文字（如果是文字点击）

## 工作流程

1. 查找目标文字或坐标
2. 计算点击位置（文字使用中心点）
3. 执行点击操作
4. 返回结果

## 注意事项

- 文字查找使用 OCR，可能不准确
- 如果有多个匹配，会点击第一个
- 建议先用 /see 确认屏幕状态
- 重要操作前请谨慎测试
